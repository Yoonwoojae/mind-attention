## 📂 성인 ADHD CBT 훈련 48개 – 실습·기록·피드백

---

### **카테고리 1. 행동 활성화 (Behavioral Activation)**

**1. 아침 루틴 만들기**

* **실습:** 루틴에 포함할 행동 3\~5개 선택 → 순서 정하기 → 알람·시각적 단서 설정
* **기록:** 실행 여부 체크, 시작·완료 시간, 기분·집중도 기록
* **피드백:** 주간 실행률 그래프, 취약 요일·시간 분석, 루틴 수정 제안

**2. 하루 시작 전 짧은 운동 습관**

* **실습:** 운동 종류(걷기·스트레칭 등)·시간 선택 → 음악·타이머 세팅
* **기록:** 운동 여부·소요 시간·운동 강도(자각척도) 기록
* **피드백:** 주간 운동 빈도·시간 그래프, 강도 조절 제안

**3. 하루 목표 3가지 설정하기**

* **실습:** 아침에 그날 꼭 해야 할 3가지 목표 입력 → 우선순위 지정
* **기록:** 완료 여부·완료 시간·목표별 어려움 정도
* **피드백:** 목표 달성률·달성 패턴 분석, 미완료 원인 피드백

**4. 긍정적인 자기 대화로 시작하기**

* **실습:** 아침 긍정 문장 1\~2개 선택 또는 작성 → 소리 내어 읽기
* **기록:** 실행 여부·당일 기분·자기확신 점수
* **피드백:** 긍정 문장 실행률, 기분 변화 추세, 새로운 문장 추천

**5. 완료 경험 쌓기(작은 과제부터 시작)**

* **실습:** 5분 이내 끝나는 과제 선택 → 즉시 실행
* **기록:** 과제 완료 여부·소요 시간·만족도
* **피드백:** 주간 완료 건수·소요 시간 통계, 난이도 조정 안내

**6. 환경 단서 활용해 행동 유도하기**

* **실습:** 행동을 유도할 시각·청각·촉각 단서 설정
* **기록:** 단서 인지 여부·행동 실행 여부 기록
* **피드백:** 단서 반응률·효과 분석, 단서 변경 제안

---

### **카테고리 2. 시간 관리 (Time Management)**

**7. 타임블로킹 일정 계획**

* **실습:** 하루 일정 블록 단위로 설정 → 각 블록 목적 입력
* **기록:** 계획 대비 실제 실행률·시간 초과 여부
* **피드백:** 계획·실행 일치율 분석, 블록 길이 최적화 제안

**8. 포모도로 집중 세션**

* **실습:** 25분 집중 + 5분 휴식 세션 수 지정
* **기록:** 세션 완료 횟수·중단 원인 기록
* **피드백:** 세션 성공률·중단 패턴 분석, 집중 시간 조정

**9. 시간 추적 기록하기**

* **실습:** 앱 타이머로 활동별 시간 측정 시작
* **기록:** 활동별 소요 시간·집중도 점수
* **피드백:** 주간 시간 사용 분석, 시간 낭비 활동 경고

**10. 마감 전 중간 체크 타이머 설정**

* **실습:** 마감 하루 전·3시간 전 알림 설정
* **기록:** 알림 반응 여부·작업 진척도
* **피드백:** 알림 효과 분석, 중간 점검 시점 조정

**11. 아침 첫 업무 ‘골든 타임’ 활용**

* **실습:** 아침 2시간 동안 집중할 최우선 업무 선택
* **기록:** 완료 여부·집중 시간·방해 요인
* **피드백:** 골든 타임 성과 분석, 방해 요인 제거 제안

**12. 주간 계획과 리뷰 루틴**

* **실습:** 매주 월요일 계획·금요일 리뷰 작성
* **기록:** 계획 대비 달성률·리뷰 메모
* **피드백:** 계획·리뷰 일치율 분석, 개선 포인트 제안

---

### **카테고리 3. 조직화·정리 (Organization Skills)**

**13. 데스크·작업공간 정리**

* **실습:** 책상 위 물건 정리·불필요 품목 제거
* **기록:** 정리 소요 시간·만족도
* **피드백:** 정리 빈도·유지 기간 분석, 개선 제안

**14. 디지털 파일·이메일 정리**

* **실습:** 폴더 구조·태그 설정 → 10분 정리
* **기록:** 삭제·정리 파일 수
* **피드백:** 파일 정리 주기·효과 분석

**15. To-do 리스트 카테고리화**

* **실습:** 할 일 분류(업무·가정·개인)
* **기록:** 카테고리별 완료 건수
* **피드백:** 완료율이 낮은 카테고리 개선 제안

**16. 한 번에 하나만 꺼내 사용하기**

* **실습:** 작업 도구 하나만 꺼내 사용
* **기록:** 규칙 준수 여부·방해 빈도
* **피드백:** 규칙 준수율, 다중 작업 감소 효과 분석

**17. 물건 위치 고정 시스템**

* **실습:** 물건 보관 위치 지정
* **기록:** 위치 유지 여부·찾는데 걸린 시간
* **피드백:** 위치 유지 성공률·탐색 시간 단축 분석

**18. 하루 마무리 정리 습관**

* **실습:** 퇴근·취침 전 10분 정리
* **기록:** 실행 여부·정리 만족도
* **피드백:** 정리 실행률·다음날 준비도 분석


### **카테고리 4. 집중 유지 (Attention Control)**

**19. 주의 분산 요인 차단(앱 차단기)**

* **실습:** 방해 앱·웹사이트 차단 목록 설정
* **기록:** 차단 실행 여부·우회 시도 횟수
* **피드백:** 차단 유지율·우회 패턴 분석, 추가 차단 제안

**20. 소음·환경 제어(화이트노이즈 등)**

* **실습:** 집중할 때 사용할 소리·환경 세팅 선택
* **기록:** 사용 시간·집중도 점수
* **피드백:** 환경별 집중도 비교, 추천 환경 제안

**21. ‘집중 모드’ 의식적 진입 절차**

* **실습:** 집중 시작 전 3단계 루틴 실행(예: 책상 정리 → 알림 끄기 → 타이머 설정)
* **기록:** 루틴 실행 여부·집중 지속 시간
* **피드백:** 루틴 실행률·집중 시간 상관 분석

**22. 시각적 집중 타겟 만들기**

* **실습:** 시선 고정 대상 또는 포커스 포인트 설정
* **기록:** 사용 여부·집중 유지 시간
* **피드백:** 포커스 사용 전후 집중도 변화 분석

**23. 집중 피로 감지·휴식 타이밍 조절**

* **실습:** 피로감 기준(예: 집중도 5 이하) 설정 후 휴식 시작
* **기록:** 휴식 전 집중도·휴식 후 회복도
* **피드백:** 휴식 빈도·효과 분석, 최적 휴식 주기 제안

**24. 단계별 난이도 조정 과제**

* **실습:** 과제를 난이도별로 쪼개서 순차 진행
* **기록:** 단계별 완료 시간·만족도
* **피드백:** 난이도 조정 전후 성취율 비교

---

### **카테고리 5. 인지 재구성 (Cognitive Restructuring)**

**25. 자동 부정 사고 기록·도전**

* **실습:** 부정적 생각 발생 시 상황·생각·대안 기록
* **기록:** 하루 기록 건수·감정 강도
* **피드백:** 부정 사고 빈도·감정 강도 변화 분석

**26. 실패 경험 재해석 훈련**

* **실습:** 실패 사건 작성 → 배운 점 2가지 이상 작성
* **기록:** 재해석 시 기분 변화
* **피드백:** 재해석 습관화 여부·정서 회복 속도 분석

**27. 완벽주의 완화 훈련**

* **실습:** 80% 완성 목표 설정 후 제출
* **기록:** 완성도·제출 만족도
* **피드백:** 80% 제출률·스트레스 감소 효과 분석

**28. 자기비난→학습 포인트 전환**

* **실습:** 자기비난 문장을 학습 포인트 문장으로 바꾸기
* **기록:** 전환 성공률·기분 변화
* **피드백:** 전환 습관화 여부·긍정 사고 증가 분석

**29. 불안한 미래 예측 수정**

* **실습:** 부정적 미래 예측 작성 → 현실 가능성 평가
* **기록:** 가능성 점수·대안 행동
* **피드백:** 예측 현실화 비율·불안도 변화 분석

**30. 과거 비교 대신 현재 기준 설정**

* **실습:** 과거·현재 상태 비교 대신 현재 목표 작성
* **기록:** 목표 달성도·기분 점수
* **피드백:** 현재 기준 목표 성취율·만족도 분석

### **카테고리 6. 감정 조절 (Emotional Regulation)**

**31. 긴장 완화 호흡법**

* **실습:** 4-7-8 호흡법 타이머 따라하기
* **기록:** 전·후 긴장도 점수
* **피드백:** 긴장 완화 평균 효과·권장 시간 제안

**32. 감정 라벨링 훈련**

* **실습:** 현재 감정을 1\~2단어로 명확히 표기
* **기록:** 하루 감정 기록 건수·감정 종류
* **피드백:** 감정 인식률·특정 감정 빈도 분석

**33. 분노 완화 대화 스크립트**

* **실습:** 분노 상황에서 사용할 문장 연습
* **기록:** 사용 여부·상황 결과
* **피드백:** 스크립트 사용 성공률·갈등 감소 분석

**34. 급격한 감정 변화 모니터링**

* **실습:** 기분 변화 알림 설정 후 변화 시 기록
* **기록:** 변화 횟수·변화 원인
* **피드백:** 변화 패턴·방아쇠 요인 분석

**35. 기분 전환 활동 리스트 만들기**

* **실습:** 즉시 가능한 기분 전환 활동 5개 작성
* **기록:** 활동 실행 여부·효과 점수
* **피드백:** 활동별 효과 순위·추천 리스트 업데이트

**36. 감사 일기 작성**

* **실습:** 하루 감사한 일 3가지 기록
* **기록:** 감사 빈도·기분 점수
* **피드백:** 감사 기록 지속률·긍정 기분 변화 분석

### **카테고리 7. 문제 해결 (Problem Solving)**

**37. 문제 정의–목표 설정 구조화**

* **실습:** 문제 상황·목표·기준 작성
* **기록:** 목표 명확성 점수
* **피드백:** 명확성 점수 추이·목표 달성률 분석

**38. 대안 아이디어 3개 이상 도출**

* **실습:** 문제당 대안 3개 이상 작성
* **기록:** 대안 수·실행 가능성 점수
* **피드백:** 대안 실행률·성공률 분석

**39. 실행 가능성 평가표 활용**

* **실습:** 대안별 장단점·실행 가능성 평가
* **기록:** 최고 점수 대안 선택 여부
* **피드백:** 선택 정확도·성공률 비교 분석

**40. 실패 대비 플랜 B 세우기**

* **실습:** 주요 계획별 대체안 작성
* **기록:** 플랜 B 실행 여부
* **피드백:** 플랜 B 성공률·위기 대응 속도 분석

**41. 복잡한 문제 쪼개기**

* **실습:** 문제를 3\~5단계로 세분화
* **기록:** 단계별 완료 여부·소요 시간
* **피드백:** 세분화 효과·완료 속도 분석

**42. ‘다음 행동’ 5분 규칙**

* **실습:** 문제 상황에서 다음 행동을 5분 안에 착수
* **기록:** 착수 시간·완료율
* **피드백:** 즉시 착수 습관화 여부 분석

### **카테고리 8. 자기 모니터링·습관 추적 (Self-monitoring & Habit Tracking)**

**43. 하루 행동 기록(시간·장소·감정)**

* **실습:** 하루 주요 행동·장소·기분 기록
* **기록:** 총 기록 건수·행동별 패턴
* **피드백:** 패턴 분석·개선 포인트 제안

**44. 주간 습관 성공률 체크**

* **실습:** 습관별 목표·실행 횟수 입력
* **기록:** 실행률·성취도
* **피드백:** 주간 평균 성공률·약점 습관 개선 제안

**45. 성취·만족도 주간 리뷰**

* **실습:** 주간 성취·만족도 점수 작성

설명 예시


설명이 너무짧잖아 아래 예시줄게

1. 문제 인식

"지갑이나 열쇠를 어디에 뒀지?"와 같은 상황이 잦다면, 여기서 제안하는 방법이 도움될 수 있다고 함.

ADHD 환자들은 부주의와 산만함 때문에 물건을 잃어버리는 경우가 일반인보다 많음.

연구에 따르면 기억력은 현재 얼마나 주의가 분산되어 있는지와 밀접하게 관련됨.


2. 원인 예시

폰을 내려놓을 때 주의가 다른 데로 가 있으면, 나중에 어디에 뒀는지 기억 못하게 됨.

작성자는 과거에 공연 티켓을 자주 잃어버렸다고 함.


3. 해결 방법
① Slow Mo Mind

'Monkey Mind'의 반대 개념으로, 서두르지 않고 천천히 중요한 물건을 제자리에 두는 습관.

열쇠, 폰, 지갑 등 자주 쓰는 물건을 둔 곳을 의도적으로 기억.


② 지정된 장소 만들기

현관 근처에 바구니나 ‘런칭패드’를 두고 지갑, 열쇠, 폰 등을 항상 그곳에 둠.


③ 80% 규칙

모든 경우를 다 대비할 수 없으니 80% 정도 성공하는 것을 목표로 함.


④ 물리적 추적 장치 활용

Apple AirTag 같은 기기를 열쇠, 지갑 등에 부착해 분실 시 ‘Find My Device’로 찾기.


⑤ 눈에 잘 띄는 색상 활용

어두운 곳에서 물건을 잃었을 때 찾기 쉽게 흰색이나 밝은 색 케이스, 지갑을 사용.


4. 결론

물건의 ‘고정된 자리’를 만들고, 이동할 때(방→방, 실내→실외) 의식적으로 속도를 늦추고 주의 깊게 물건을 제자리에 두는
