# **home 탭 기능 요구사항**

## **1. 현재 모듈 진행상황 (상단)**

### **화면 구성**
- "You're on chapter X of Y"
- 모듈명 표시 (예: "Raising low self-esteem")
- 현재 세션 진행률 (0%, 25%, 50%, 75%, 100%)
- 진행률 바

### **기능**
- 현재 학습 중인 모듈의 몇 번째 세션인지 표시
- 해당 세션 내 완료도 표시
- 세션 완료 시 다음 세션으로 자동 이동

---

## **2. Today's exercises**

### **화면 구성**
- "Today's exercises" 제목
- "View module" 링크
- 현재 세션의 항목들 (완료된 것만 표시)

### **기능**
- **현재 진행중인 세션의 항목들을 보여줌**
- **1개만 완료**: 그 1개만 표시
- **2개 이상 완료**: 완료된 항목들만 표시
- **현재 진행중카드가 상단에 크게 보이고 하단에 작게 완료된 애들 보여짐**
- **클릭 시**: 해당 세션/레슨 화면으로 이동
- "View module" 클릭 시 전체 모듈로 이동

---

## **3. Weekly learning goal**

### **화면 구성**
- "Weekly learning goal" 제목
- 설정된 목표 진행상황 (X/Y 형식)
- 일별 체크 표시

### **기능**
- **최초 설정**: "이번주는 몇 번 참석 가능한가요?" 질문
- 선택 옵션: 1~7일 중 선택
- **매일 접속/활동 시 자동으로 1개씩 체크**
- 주간 목표 달성률 시각화
- 주 단위로 리셋

---

## **4. Daily focus**

### **화면 구성**
- "Daily focus" 제목
- 미설정 시: "What's your focus today?" 버튼
- 설정 완료 시: 목표 텍스트 표시

### **기능 플로우**

#### **Step 1: 목표 입력**
- "What's your focus today?" 화면
- 텍스트 입력 필드 (90자 제한)
- "Continue" 버튼

#### **Step 2: 알림 설정**
- "Set a reminder" 화면
- **시작 알림**: 시간 설정
- **끝 알림**: 시간 설정
- "Save" 버튼

#### **완료 후**
- 설정된 목표 표시
- 수정/삭제 가능
- 설정한 시간에 푸시 알림 발송

---

## **5. 전체 동작 흐름**

1. **모듈 진행**: 세션별로 순차 진행
2. **Today's exercises**: 현재 세션에서 완료한 항목들만 표시
3. **Weekly goal**: 일일 접속 시 자동 체크
4. **Daily focus**: 하루 1회 설정, 알림 기능 포함

- 완료된 모듈 3개 표시 (최신순)

#### **기능**
- **최신 완료 순서로 3개만 표시**
- 각 모듈별 완료 날짜 표시
- 모듈 썸네일 및 제목

  - 전체 완료 모듈 목록 화면으로 이동
  - 스크롤 가능한 리스트 형태

- 완료된 모듈 클릭 시 복습 모드로 접근

## **2. All modules 탭**

### **Featured 섹션**

- 추천/인기 모듈 1개 표시
- 각 모듈별 썸네일, 제목, 간단한 설명

#### **기능**

#### **기능**
- 8개 주제 카테고리 표시
- 각 카테고리별 색상 구분
- 카테고리 클릭 시 해당 주제의 모듈 목록으로 이동
- 각 카테고리별 모듈 개수 표시 (선택사항)

### **Recommended 섹션**
#### **화면 구성**
- "Recommended" 제목
- 개인화된 추천 모듈들
- 추천 이유 간단 표시

#### **기능**
- **사용자 맞춤 추천 알고리즘**:
  - 완료한 모듈 기반
  - 관심 주제 기반
  - 학습 패턴 분석
- 모듈별 추천 이유 표시 ("Based on your interest in...")
- 추천 모듈 클릭 시 상세 정보

# **Focus 탭 기능 요구사항**

## **개요**
ADHD 실생활 극복을 위한 과학적으로 검증된 도구 모음

## **1. ADHD 특화 타이머 (포모도로 + 시각적)**

### **기능 설명**
- 포모도로 기법 기반의 집중 타이머
- ADHD 사용자를 위한 짧은 시간 단위 옵션
- 시각적 시간 표시로 시간 맹목성 극복

### **주요 기능**
- **시간 설정**: 10분, 15분, 20분, 25분 선택
- **시각적 표시**: 남은 시간을 색상/원형 그래프로 표시
- **휴식 시간**: 5분 자동 휴식 알림
- **진동/소리 알림**: 집중 시작/종료 알림
- **통계**: 일일 집중 시간 누적 기록

---

## **2. 작업 분해 도구 (CBT 기반)**

### **기능 설명**
- 큰 작업을 작은 단위로 나누어 ADHD 마비 현상 방지
- CBT(인지행동치료) 원리 적용

### **주요 기능**
- **작업 입력**: 해야 할 일 입력
- **자동 분해**: AI가 2-3분 단위로 자동 분해
- **수동 조정**: "더 작게" 버튼으로 추가 분해
- **체크리스트**: 완료 체크로 성취감 제공
- **예상 시간**: 각 작업별 소요 시간 표시

---

## **3. 마음챙김/집중 보조 도구 (MBCT 기반)**

### **기능 설명**
- 짧은 마음챙김 운동으로 주의력 재설정
- MBCT(마음챙김 기반 인지치료) 원리 적용

### **주요 기능**
- **1분 호흡**: 가이드 호흡 운동
- **3분 바디스캔**: 신체 감각 인식 운동
- **주의력 리셋**: 30초 집중 운동
- **타이머 내장**: 운동별 자동 타이머
- **기록**: 일일 실천 횟수 기록

---

## **4. 환경 구조화 도구**

### **기능 설명**
- 집중에 최적화된 환경 만들기
- 외부화 전략으로 실행 기능 보조

### **주요 기능**
- **환경 체크리스트**: 집중 환경 점검 항목
- **방해 요소 관리**: 제거해야 할 항목 목록
- **맞춤 설정**: 개인별 체크리스트 저장
- **루틴 생성**: 집중 전 준비 루틴 만들기
- **알림 설정**: 환경 점검 시간 알림

---

## **5. 공통 기능**

### **데이터 관리**
- 모든 활동 기록 자동 저장
- 주간/월간 통계 제공
- 진전 상황 시각화

### **접근성**
- 간단한 UI/UX
- 큰 버튼과 명확한 피드백
- 최소한의 단계로 기능 실행


# **Profile 탭 기능 요구사항**

## **1. 프로필 섹션 (상단)**

### **화면 구성**
- 프로필 아바타 (원형, 이니셜 표시)
- 사용자명 표시 (예: "yoon")
- "Edit profile" 링크
- 우상단 설정 아이콘 (톱니바퀴)

### **기능**
- **아바타**: 이니셜 자동 생성 또는 사진 업로드
- **사용자명**: 현재 설정된 이름 표시
- **"Edit profile" 클릭 시**: 프로필 편집 화면으로 이동
- **설정 아이콘 클릭 시**: 설정 화면으로 이동

---

## **2. 통계 섹션 (3개 지표)**

### **화면 구성**
- 3개 원형 아이콘과 숫자
- **54 exercises completed** (별 아이콘)
- **1 weekly streak** (물방울 아이콘)
- **0 community posts** (웃는 얼굴 아이콘)

### **기능**
- **exercises completed**: 완료한 총 연습/세션 수 실시간 카운트
- **weekly streak**: 주간 연속 접속/활동 기록
- **community posts**: 0으로 고정 (커뮤니티 기능 없음)
- 각 지표 클릭 시 상세 히스토리 보기 (선택사항)

---

## **3. Your toolkit 섹션**

### **화면 구성**
- "Your toolkit" 제목
- 3개 카드 형태의 항목들

### **항목별 구성**

#### **Key ideas**
- 아이콘: 별이 있는 보라색 카드
- 제목: "Key ideas"
- 설명: "Key takeaways from each lesson for whenever you need a refresher"

#### **Brain hacks**
- 아이콘: 뇌 모양이 있는 민트색 카드
- 제목: "Brain hacks"
- 설명: "Simple tips and tricks to make everyday tasks easier"

#### **Reflections**
- 아이콘: 전구가 있는 보라색 카드
- 제목: "Reflections"
- 설명: "A collection of your reflections to track your progress over time"

### **기능**
- **Key ideas 클릭**: 학습한 레슨들의 핵심 요점 모음 화면
- **Brain hacks 클릭**: 실용적인 팁과 트릭 모음 화면
- **Reflections 클릭**: 개인 성찰 기록 및 진행상황 추적 화면
- 각 섹션별 콘텐츠 저장 및 관리
- 검색 기능 (키워드로 찾기)

---

## **4. 프로필 편집 기능**

### **Edit profile 화면**
- 프로필 사진 변경 ("Edit photo")
- 이름 입력 필드
- 사용자명 입력 필드
- 바이오 입력 (250자 제한)
- 위치 입력 필드
- "Hide profile" 토글 스위치
- "Save" 버튼

---

## **5. 설정 기능**

### **Settings 화면**
- **Notifications**: 알림 설정 상세 화면
- **Privacy Policy**: 개인정보처리방침
- **Account management & Help**: 계정 관리 및 도움말
- **Terms & Conditions**: 이용약관
- 로그인 정보 표시
- "Log out" 버튼
- "Software Information" 링크

### **Notifications 상세 설정**
- Daily Inflow reminder
- Quinn notifications
- Daily Focus reminders (입력/완료)
- Medication reminders (아침/오후/저녁)
- 각각 시간 설정 가능

---

## **6. 데이터 관리**

### **통계 데이터 추적**
- 완료한 연습 수 자동 카운트
- 주간 연속 기록 계산 및 업데이트
- 학습 패턴 분석

### **개인 콘텐츠 저장**
- Key ideas 자동 수집 및 저장
- Brain hacks 북마크 및 관리
- Reflections 입력 및 히스토리 관리

# **모듈 상세 화면 UI 구조**

## **공통 UI 구조 (모든 모듈 동일)**

### **1. 상단 헤더**
- 뒤로 가기 버튼 (좌상단)

  - 아이콘 (식물 화분 등)

6. **세션 타입 구성** (Lesson, Reflection 등의 조합이 다름)

### **동일한 부분**
1. **전체 레이아웃 구조**
2. **전문가 정보 카드 디자인**
3. **세션 카드 UI 디자인**
4. **색상 테마**
5. **버튼 스타일**
6. **폰트 및 간격**

### **상태 관리**
- 각 세션별 진행 상태 (미시작/진행중/완료)
- 전체 모듈 진행률 계산
- 잠김/해제 상태 관리

### **네비게이션**
- 세션 클릭 시 해당 세션으로 이동
- "Start module" 클릭 시 첫 번째 세션으로 이동
- 완료된 세션은 재방문 가능



훈련 화면

# **모듈 세션 타입별 화면 구조 및 기능**

## **1. Check-in**

### **존재 패턴**
- **있는 모듈**: 모듈 시작 시 1개 + 모듈 끝 시 1개 (총 2개)
- **없는 모듈**: Check-in 없이 바로 Lesson부터 시작

### **화면 구조**
- **5개 질문** 연속 진행
- 각 질문별 객관식 선택지
- 공통 선택지 패턴: "Never", "Rarely", "Sometimes", "Often", "Always" 등

### **기능**
- 설문조사 형태의 자가진단
- 모듈 시작 전후 상태 비교 가능
- 선택 완료 후 다음 질문으로 자동 이동
- 모든 질문 완료 시 다음 세션으로 이동

---

## **2. Lesson**

### **화면 구조**
- **긴 텍스트 콘텐츠** (스크롤 가능)
- **오디오 플레이어** (상단 또는 하단)
- 진행률 표시
- "Continue" 또는 "Next" 버튼

### **기능**
- **오디오 재생**: 텍스트를 음성으로 읽어줌
- **재생 컨트롤**: 재생/일시정지, 진행바, 속도 조절
- **텍스트 하이라이트**: 현재 읽고 있는 부분 강조 (선택사항)
- **오프라인 지원**: 미리 다운로드된 오디오 재생
- 학습 완료 후 자동으로 다음 세션 해제

---

## **3. Reflection**

### **화면 구조**
- **질문 또는 주제** 제시
- **텍스트 입력 필드** (여러 줄 가능)
- "Skip" 버튼
- "Continue" 버튼

### **기능**
- **개인 성찰 질문** 제시
- **자유 텍스트 입력** (글자 수 제한 없음 또는 넉넉한 제한)
- **선택적 작성**: 안 적고 Skip 가능
- **저장 기능**: 작성 내용 자동 저장
- **Profile > Reflections**에서 나중에 다시 볼 수 있음

---

## **4. Brain hack**

### **화면 구조**
- **CBT 도구/기법 제목**
- **상세 설명 텍스트**
- **사용법 안내**
- **예시 또는 실습 가이드**
- "Got it" 또는 "Continue" 버튼

### **기능**
- **CBT 기반 실용 도구** 소개
- **단계별 사용법** 설명
- **실생활 적용 예시** 제공
- **북마크 기능**: Profile > Brain hacks에 자동 저장
- 도구 활용법 학습 완료 후 다음 단계 진행

---

## **5. Quiz**

### **화면 구조**
- **질문 텍스트**
- **객관식 선택지** (보통 3-4개)
- 질문 번호 표시 (1/3, 2/3 등)
- 선택 후 "Continue" 버튼

### **기능**
- **모듈 내용 이해도 확인**
- **즉시 피드백**: 정답/오답 표시
- **해설 제공**: 왜 그 답이 맞는지 설명
- **재시도 가능**: 틀려도 다시 선택 가능
- 모든 문제 완료 시 점수 또는 완료 메시지

---

## **공통 진행 방식**

### **세션 간 연결**
1. Check-in (있는 경우) → Lesson → Reflection → Brain hack → Quiz → Check-in (마지막)
2. 각 세션 완료 시 다음 세션 자동 해제
3. 중간에 나가도 진행상황 저장

### **진행상황 관리**
- 각 세션별 완료 상태 추적
- Today 탭에서 완료된 세션들만 표시
- 전체 모듈 진행률 계산
- Profile 탭 통계에 반영

### **공통 UI 요소**
- 상단 진행률 바
- 뒤로 가기 버튼
- 세션 타입 아이콘 및 제목
- 하단 액션 버튼 (Continue, Skip, Next 등)

### **데이터 저장**
- **Lesson**: 완료 여부만 저장
- **Reflection**: 작성 내용 저장 (Profile에서 조회 가능)
- **Brain hack**: 학습한 도구 목록 저장 (Profile에서 조회 가능)
- **Quiz**: 정답률 및 완료 여부 저장
- **Check-in**: 응답 결과 저장 (진전 추적용)
