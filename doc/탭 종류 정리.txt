## ✅ \[1단계] MVP – 하단 탭 3개 구성

| **하단 탭 이름**               | **주요 기능**                                                                   | **목적**                   |
| ------------------------- | --------------------------------------------------------------------------- | ------------------------ |
| **Home**                  | - 오늘의 감정 상태 요약<br>- 오늘의 훈련 추천<br>- 최근 훈련/진행 요약<br>- 퀵 액션 버튼 (CBT/루틴 진입)<br> | 앱 전체 허브 / 감정기록 연동 포함     |
| **CBT**                   | - 자동사고 탐지<br>- 감정 유발 상황 구조화<br>- 사고 재구성 훈련<br>- 일일 미션형 세션 콘텐츠<br>           | 인지 기반 증상 완화 (핵심 콘텐츠)     |
| **Focus**<br>(또는 Routine) | - 루틴 생성기<br>- 할 일 분할 도구<br>- 집중 타이머 (Pomodoro)<br>- 미루기 방지 미션<br>           | 실행기능 개선 (계획, 실행력, 집중력 등) |

---

## ✅ \[2단계] 확장 – Calm 탭 추가 (4개 탭)

| **하단 탭 이름** | **주요 기능**                                                                      | **목적**                |
| ----------- | ------------------------------------------------------------------------------ | --------------------- |
| **Calm**    | - 짧은 명상 (3\~5분)<br>- 바디스캔, 감각 인식 훈련<br>- 시각화된 숨쉬기 가이드<br>- 명상 전/후 감정 변화 체크<br> | 정서 안정 / 충동 조절 / 집중 회복 |
