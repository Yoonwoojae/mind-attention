# Focus 도구모음 v2 (근거기반 개선안)

## 공통 설계 원칙

* **원클릭 착수**: 어디서든 1탭으로 시작(타이머/리셋/분해 바로 실행).
* **Now → Next**: 지금 할 1개만 보여주고, 끝나면 다음 1개 제시.
* **짧게·가변적**: 30초\~25분 범위, 개인화된 자동 추천.
* **외부 단서 강화**: 시각적 잔여시간, 진동·소리, 체크리스트 카드.
* **미세 보상**: 완료 즉시 소규모 리워드(배지/연속일/포인트).
* **데이터 최소·로컬 우선**: 민감정보 비수집, 익명 집계만.

## 1) ADHD 타이머 + 시각화 (개선)

### 핵심

* **프리셋**: 5/10/15/20/25분 + 사용자 설정(1\~30분 슬라이더).
* **시각화**: 홈/잠금화면 위젯용 **잔여시간 링**·원형 카운트다운.
* **휴식**: 30\~90초 **움직임 휴식**(기지개·물 마시기) 자동 제시.
* **알림**: 시작/종료/막바지(남은 1분) 하프틱·사운드.
* **실패 안전장치**: 앱이 백그라운드 가면 **자동 일시정지** 옵션.

### 추가(권장)

* **동적 길이 추천**: 캘린더 ‘다음 약속까지 남은 시간’ 기반 추천(예: “지금 12분 가능”).
* **방해 차단**: DND/알림 차단 토글(가능한 OS 통합).
* **즉시보상**: 블록 종료 시 ‘오늘 n블록’ 애니메이션+스몰 보상.

### 로깅(요약)

`session_id, start_ts, planned_min, actual_min, break_used, paused_count, aborted(bool)`

## 2) 작업 분해 도구 (CBT/MCT 핵심 반영)

### 흐름

**Capture(무엇?) → Chunk(쪼개기) → Commit(하나만 약속) → Do(타이머 연결) → Review(짧은 회고)**

### 기능

* **빠른 입력**: 텍스트·음성·공유시트로 “큰 일” 캡처.
* **AI 초안 분해**: 2\~10개 소단계 제안 → **“더 작게”** 반복 버튼.
* **가변 길이**: 각 소단계에 **예상 3/5/10/15분** 태그.
* **에너지·컨텍스트 태그**: 뇌피로/소음/자리 필요 여부 체크.
* **한 개만 표시**: ‘지금 할 1개’만 카드로 고정 → 완료 후 다음 카드 노출.
* **안티-완벽주의 넛지**: “초안 80%면 넘어가기” 토글.

### 템플릿(예시)

* 이메일 회신, 장보기 준비, 보고서 초안, 청소 10분 스프린트 등 **자주 쓰는 템플릿** 저장.

### 로깅

`task_id, step_count, step_est_min[], actual_min[], rechunk_count`

## 3) 집중 리셋 (MBI 보조·초단기 모듈 최적화)

### 빠른 액션

* **SOS 60초**: 20초 호흡(시각·진동 페이서) → 20초 스트레칭 → 20초 “지금 할 1개 선택”.
* **1/3/5/10분 모듈**: 눈 뜨고 가능한 호흡·바디스캔·주의전환.
* **노이즈 타이머**: 화이트/핑크/브라운 노이즈 5·10·20분.

### 피드백

* **전/후 슬라이더 2개**: 집중도, 긴장도(0\~5). 변화만 저장.

### 로깅

`module_type, planned_min, actual_min, pre_focus, post_focus`

---

## 4) 환경 구조화 (가장 강하게 권장)

### 포커스 레시피

* **장소별 레시피**: 집/사무실/카페 프로필별 **체크리스트+원터치 액션** 묶음.

  * 예: DND 켜기, 밝기 조정, 헤드폰 안내, 탭 정리, 타이머 10분 시작.
* **2-Min Cleanup**: 시작 전 **책상정리 2분 카운트다운**.
* **방해요소 블랙리스트**: 자주 새는 앱/웹을 목록화 → 세션 중 리마인더.
* **시작/종료 루틴**: 시작 3항목, 종료 3항목(정리·다음 한 개 예약).

### 로깅

`recipe_id, checklist_done%, distractions_flagged[]`

### Implementation Intentions (If-Then 플래너)

* **형식**: “만약 \[상황]이면, 나는 \[구체 행동]을 하겠다.”
* **예시**: “만약 14:00 알림이 울리면, 5분 타이머로 이메일 1개만 처리.”
* **연결**: 저장 시 해당 타이머/레시피와 자동 연결.
