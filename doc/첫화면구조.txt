화면 구조를 정해야 하는데
1. intro
2. 로그인 체크해서 되어있으면 바로 홈, 안되어있으면 onboarding( 간략한 설명 4-5가지를 옆으로 넘기면서 볼수있게 설정하고 맨 하단 로그인, 회원가입 버튼 고정)
3. 로그인버튼누르면 로그인화면 회원가입 버튼 누르면 회원가입 화면
4. 회원가입 완료하면 설문조사화면
5. 설문조사 끝나면 분석화면으로 넘어가서 분석중입니다 표시.
6. 분석끝나면 어떤 플랜이 어울리는지 3개 추천.
7. 다음 누르면 알림 설정화면에서 알림설정.( 선택 )
8. 설정완료하면 이 앱을 통해 뭘 얻을수 있는지 간략한 설명.
9. 다음 누르면 무료 기간중에 어떻게 알림으로 안내할건지 설명.(푸쉬)
10. 다음 누르면 7일 무료 설명과 함께 1년, 1달 금액과 설명, 하단에 사람들 리뷰 몇개 오른쪽으로 스와이프해서 볼수있게.
11. 결제하기 누르면 구글 플레이스토어 결제창 띄워주고 결제. ( 여기에서 결제 안하면 다음으로 못넘어감. 무조건 결제 해야 이용가능하고 어차피 7일 무료기간있기에 괜찮을듯)
12. 결제 완료하면 사용자 맞춤 설정:
    - 목표 선택: 스트레스 해소 / 집중력 향상 / 감정 조절 / 수면 개선
    - 경험 수준: 처음이에요(입문) / 가끔 해봤어요(초급) / 꾸준히 하고 있어요(중급)
    - 선호 스타일: 짧고 간단하게(5-10분) / 차근차근 깊이있게(15-20분)
13. 선택값에 따라 맞춤 로드맵 자동 생성 후 간략하게 표시.
14. 다음 누르면 바로 첫번째 레슨 시작.
15. 첫번째 레슨 화면에서 뒤로가기하면 홈 화면으로.

온보딩

1번째 화면:
  - 제목: 집중력 향상
  - 설명: 짧고 효과적인 명상으로 산만함을 줄이고 집중력을 높여요

  2번째 화면:
  - 제목: 감정 조절
  - 설명: 충동적인 감정을 다스리고 차분한 마음을 유지해요

  3번째 화면:
  - 제목: 수면 개선
  - 설명: 과도한 생각을 멈추고 편안한 수면을 취해요

  4번째 화면:
  - 제목: 맞춤형 알림
  - 설명: 당신의 일정에 맞춰 적절한 시간에 알려드려요

  5번째 화면:
  - 제목: 과학적 접근
  - 설명: ADHD 전문가들과 함께 개발한 검증된 프로그램
